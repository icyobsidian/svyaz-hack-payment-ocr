# Jupyter Notebooks

Демонстрационные ноутбуки для проекта распознавания платежных счетов.

## Ноутбуки

1. **pipeline_demo.ipynb** - Демонстрация работы пайплайна обработки PDF
2. **model_training.ipynb** - Обучение ML модели (если используется)

## Установка

```bash
pip install jupyter notebook
pip install -r ../backend/requirements.txt
```

## Запуск

```bash
jupyter notebook
```
