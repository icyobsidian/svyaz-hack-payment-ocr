# Changelog

## [1.0.0] - 2025-11-15

### Добавлено

#### Основной функционал
- ✅ Telegram бот на aiogram для загрузки PDF файлов
- ✅ Backend API на FastAPI для обработки PDF
- ✅ Парсинг PDF (PyMuPDF, pdfplumber)
- ✅ OCR обработка (Tesseract) для сканированных документов
- ✅ Извлечение структурированных данных из платежных счетов
- ✅ Docker контейнеризация всех сервисов
- ✅ Jupyter notebooks для демонстрации

#### ML и улучшения
- ✅ ML модель для извлечения полей (TF-IDF + Logistic Regression)
- ✅ Система обучения модели через API
- ✅ Кэширование результатов обработки
- ✅ Метрики производительности
- ✅ Расширенный список извлекаемых полей

#### Извлекаемые поля
- Основные: номер счета, дата, сумма
- Плательщик: наименование, ИНН, КПП
- Получатель: наименование, ИНН, КПП
- Банковские реквизиты: БИК, р/с, наименование банка
- Дополнительные: валюта, НДС, сумма прописью, договор, срок оплаты
- Назначение платежа

#### API Endpoints
- `POST /api/v1/process-pdf` - обработка PDF
- `POST /api/v1/train-model` - обучение модели
- `GET /api/v1/performance` - метрики производительности
- `POST /api/v1/performance/reset` - сброс метрик
- `GET /health` - health check

### Технологии

- **Backend**: FastAPI, Python 3.11
- **Telegram Bot**: aiogram 3.13
- **PDF Processing**: PyMuPDF, pdfplumber
- **OCR**: Tesseract OCR
- **ML**: scikit-learn, TF-IDF, Logistic Regression
- **Кэширование**: In-memory LRU cache
- **Контейнеризация**: Docker, Docker Compose

### Производительность

- Кэширование результатов обработки
- Автоматическое переключение между парсингом и OCR
- Метрики времени обработки
- Оптимизация через комбинацию ML и regex
